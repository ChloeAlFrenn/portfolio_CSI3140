Hotel Reservation Website

As part of the Database 1 course at the University of Ottawa, our team of three members designed and developed a hotel reservation website. This project integrated a PostgreSQL database to manage and retrieve reservation data and utilized modern web technologies for a seamless user experience. The front end was built with React, providing an intuitive interface for users to book hotel rooms, while the back end was developed with Java and Spring Boot, ensuring efficient handling of requests. It includes the following technologies:

  • React: Used to build the dynamic and responsive front-end user interface.
  • Java 19: Served as the primary programming language for the back end, offering a reliable and high-performance environment for server-side logic.
  • Spring Boot: simplified the development of the back end, providing various features like embedded servers and simplified dependency management.
  • PostgreSQL: Chosen for the database layer, PostgreSQL offered robust and scalable database management capabilities, ensuring efficient data storage and retrieval.
  • Maven: Used for managing project dependencies and building the Java-based back end, ensuring a consistent and reproducible build process.
  • Node.js:: Essential for running the front-end development environment, facilitating the installation and management of dependencies and serving the React application during development.

Tic Tac Toe Game

Developed in a team of 2 as a single-player game for the CSI3140 course at the University of Ottawa, this Tic Tac Toe game showcases proficiency in HTML, CSS, PHP, AJAX, JSON, and JavaScript. The game consists of a 3x3 grid where the player and the computer take turns clicking squares to place their mark (X for the player, O for the computer). The game indicates whose turn it is.

  • HTML, CSS, JavaScript: Used to create the game interface and implement game logic.

  • PHP, AJAX, JSON: Used to store the game state and manage the leaderboard on a server. The JavaScript logic communicates with the PHP server by making AJAX calls to the api.php file and by sending and receiving data through JSON files

Food order web application

Developed in a team of 2 using a database and the approaches learnedin the CSI3140 course. at the University of Ottawa, this food order web application showcases proficiency in HTML, CSS, PHP, AJAX, JSON, JavaScript and PostgreSQL. This application consists of a web interface for clients to place orders which will then be saved in the database. An admin can view and delete orders through the web interface.

  • PostgreSQL: Used as the database management system for storing order data and item information.
  • PHP: Implemented for server-side scripting and handling database interactions.
  • AJAX: Enabled asynchronous communication between the client and server for a smoother user experience.
  • JavaScript: Used to handle client-side logic and dynamic interactions on the web pages.
  • HTML & CSS: Utilized to structure and style the web application's interface.
