Hotel Reservation Website

As part of the Database 1 course at the University of Ottawa, our team of three members designed and developed a hotel reservation website. This project integrated a PostgreSQL database to manage and retrieve reservation data and utilized modern web technologies for a seamless user experience. The front end was built with React, providing an intuitive interface for users to book hotel rooms, while the back end was developed with Java and Spring Boot, ensuring efficient handling of requests. It includes the following technologies:

  • React: Used to build the dynamic and responsive front-end user interface.
  • Java 19: Served as the primary programming language for the back end, offering a reliable and high-performance environment for server-side logic.
  • Spring Boot: simplified the development of the back end, providing various features like embedded servers and simplified dependency management.
  • PostgreSQL: Chosen for the database layer, PostgreSQL offered robust and scalable database management capabilities, ensuring efficient data storage and retrieval.
  • Maven: Used for managing project dependencies and building the Java-based back end, ensuring a consistent and reproducible build process.
  • Node.js:: Essential for running the front-end development environment, facilitating the installation and management of dependencies and serving the React application during development.

Tic Tac Toe Game

Developed in a team of 2 as a single-player game for the CSI3140 course at the University of Ottawa, this Tic Tac Toe game showcases proficiency in HTML, CSS, and JavaScript. The game consists of a 3x3 grid where the player and the computer take turns clicking squares to place their mark (X for the player, O for the computer). The game indicates whose turn it is.

  • HTML, CSS, JavaScript: Used to create the game interface and implement game logic.
